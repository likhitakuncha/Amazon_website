# Amazon Clone

A responsive Amazon homepage clone built using **HTML** and **CSS**. This project recreates the look and feel of Amazon's landing page to practice front-end web development concepts such as Flexbox, layouts, responsive design, and styling.

##  Features

* Amazon-style navigation bar
* Search bar with category selector
* Hero banner section
* Product category cards
* Multi-section footer
* Hover effects and clean UI
* Responsive layout using Flexbox

## 🛠️ Technologies Used

* HTML5
* CSS3
* Font Awesome
